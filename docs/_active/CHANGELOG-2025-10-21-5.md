# Changelog - 21/10/2025 (Parte 5)

## Correções no Sistema de Mapas, Exibição de Contatos e Salvamento de Imagens

### 1. Correções no Sistema de Mapas

#### a) Mapa do Diretório (`/diretorio`)

**Problema**: O mapa carregava no meio do oceano e não usava a geolocalização do usuário automaticamente. O zoom era muito aproximado.

**Solução**:
- **Geolocalização Automática**: Adicionada geolocalização automática ao carregar a página. O mapa agora centraliza na localização do usuário, se disponível.
- **Ajuste de Zoom**: O zoom foi ajustado para 11 (em vez de 12) para mostrar uma área maior e mais negócios.

**Arquivos modificados**:
- `src/pages/Diretorio.tsx`

#### b) Mapa do Perfil de Negócio

**Problema**: O mapa tinha zoom excessivo e não centralizava corretamente no endereço do negócio.

**Solução**:
- **Ajuste de Zoom**: O zoom foi ajustado para 12 (em vez de 13).
- **Centralização**: O mapa agora centraliza no endereço do negócio (latitude/longitude) quando disponível.

**Arquivos modificados**:
- `src/components/maps/BusinessLeafletMap.tsx`

### 2. Correção na Exibição de Informações de Contato

**Problema**: O bloco de "Informações de Contato" não era exibido no perfil público se não houvesse nenhum contato preenchido.

**Solução**: O card agora é sempre exibido. Se não houver contatos, ele mostra a mensagem "Entre em contato pelo formulário abaixo."

**Arquivos modificados**:
- `src/pages/DiretorioEmpresa.tsx`

### 3. Correção no Salvamento de Imagens

**Problema**: As imagens (logo, capa e galeria) eram enviadas para o Storage, mas não salvas no banco de dados até que o usuário clicasse em "Atualizar Empresa" em outra aba.

**Solução**:
- **Salvamento Imediato**: Adicionados botões "Salvar Logo" e "Salvar Capa" que salvam as imagens imediatamente após o upload.
- **Galeria Automática**: As imagens da galeria agora são salvas automaticamente no banco de dados após o upload.

**Arquivos modificados**:
- `src/pages/DashboardEmpresa.tsx`
